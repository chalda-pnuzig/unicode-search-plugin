# Unicode Search

[![Version](https://img.shields.io/jetbrains/plugin/v/30701-unicode-search.svg)](https://plugins.jetbrains.com/plugin/30701-unicode-search)
[![Downloads](https://img.shields.io/jetbrains/plugin/d/30701-unicode-search.svg)](https://plugins.jetbrains.com/plugin/30701-unicode-search)

Search and insert Unicode characters directly into your editor with ease.

This tool window plugin helps you quickly find, copy, and insert Unicode characters and symbols directly into your JetBrains IDE.

## 🚀 Features

- **Advanced Search**: Search by character name, Unicode block, category, or hex codepoint.
- **Multiple Formats**: Copy or insert characters as raw text, hex, HTML entity, decimal, or character name.
- **Visual Preview**: Detailed visual preview with hover tooltips showing code, block, and category.
- **Powerful Engine**: Supports multi-word queries, supplementary Unicode planes, and features a customizable pagination limit.
- **Quick Access**: Right-click context menu for quick copy and insertion.

## 📥 Installation

### From JetBrains Marketplace

The simplest way to install the plugin is directly from your IDE:

1. Open your IDE Settings / Preferences:
   - **Windows/Linux**: `File` > `Settings` > `Plugins`
   - **macOS**: `IDE Name` > `Preferences` > `Plugins`
2. Select the **Marketplace** tab.
3. Search for **Unicode Search** (you can also use the vendor name *Chalda*).
4. Click **Install**.
5. Restart your IDE if prompted.

Alternatively, you can visit the [JetBrains Marketplace Plugin Page](https://plugins.jetbrains.com/plugin/30701-unicode-search) to push the installation to your IDE.

### Manual Installation

1. Download the latest release `.zip` from the [Marketplace](https://plugins.jetbrains.com/plugin/30701-unicode-search).
2. Open `Settings` > `Plugins` > click on the `⚙️` (gear icon) > `Install Plugin from Disk...`.
3. Select the downloaded `.zip` archive and restart the IDE.

## 💡 Usage

Once installed, you can access the tool window directly from the IDE workspace:

1. Look for the **Unicode Search** tool window on the **right side** of your IDE.
2. Open it and start typing in the search bar to filter characters by name, hex code, block, or category.
3. **Left-click** on any character to insert it directly at your cursor's current position in the active editor.
4. **Right-click** on a character to access a context menu that allows you to copy or insert the character in various additional formats:
   - Raw Text
   - Hexadecimal
   - HTML Entity
   - Decimal
   - Character Name
